The "Effort Remaining" Metric

"Win Number" is 110% of Biden's votes in the 2020 election.

"Biden Expected Votes" for 2024 is 84% of Biden's 2020 votes.

"Biden Committed Votes" is zero at the moment. The intent is to make this quantity update dynamically in future versions.

"Effort Required" is "Win Number" minus "Expected Votes"

"Effort Remaining" is "Effort Required" minus "Committed Votes" (so currently identical to "Effort Required").


                def ComputeEffortRequired(df, win_factor = 1.10, coefficient_2024 = .84):
                    df['WinNumber2024'] = df['Biden20'] * win_factor
                    df['Expected2024'] = df['Biden20'] * coefficient_2024
                    df['VotesCommitted'] = 0
                    
                    df.fillna({'WinNumber2024': df['WinNumber2024'].mean(), 'Expected2024': df['Expected2024'].mean()}, inplace=True)
                    df['WinNumber2024'] = df['WinNumber2024'].astype(int)
                    df['Expected2024'] = df['Expected2024'].astype(int)
                    
                    df['EffortRequired'] = (df['WinNumber2024'] - df['Expected2024'])
                    df['EffortRemaining'] = df['EffortRequired'] - df['VotesCommitted']
                    
                    return df
